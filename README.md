# マイウォーキングアプリ (v10.17)

自分専用にカスタマイズされた、究極のパーソナルヘルスケア＆ウォーキングナビゲーションアプリです。
ブラウザ上で動作しつつ、iOSのショートカット機能や各種APIを駆使することで、市販の有料アプリを超える機能を実現しています。

## 🌟 最新の追加機能 (v10.17)
* **📷 AI栄養成分スキャナー搭載**
  食品パッケージの裏にある「栄養成分表示」をカメラで撮影するだけで、Gemini AIが数値を解析。カロリー・PFC（タンパク質・脂質・炭水化物）・塩分を抽出し、食事記録に**自動加算**します。（※面倒な手入力やバーコード辞書は不要！）

## 🚀 主な機能一覧

### 🚶‍♂️ ウォーキング・マップ機能 (AIナビ)
* **自動追従マップ:** Googleマップのように、歩行に合わせて現在地を自動追従。画面をスワイプすると手動操作モードにシームレスに切り替わります。
* **周回ルート自動生成:** 「3km歩きたい」「5000歩歩きたい」と入力するだけで、現在地からぐるっと回って戻ってくる最適な散歩ルートをAIが自動生成します。
* **目的地・中継地ナビ:** 地図をタップして目的地（赤ピン）や寄り道したい中継地（緑ピン）を設定し、最短ルートを検索できます。
* **全画面操作パネル:** 全画面表示中も、画面上部の専用パネルから「現在地に戻る」「中継地の追加」「ピンの消去」「ルートの再検索」が歩きながら快適に行えます。

### ⚖️ 体調・食事・カロリー管理
* **パーソナルデータ連動:** 身長・体重・年齢・性別から「基礎代謝」と「BMI」を自動計算。
* **運動消費カロリー計算:** 歩数と階数から、運動による消費カロリーを自動算出します。
* **PFCバランス記録:** 朝・昼・夕・間食ごとに詳細な数値を記録可能。
* **カロリー収支判定:** 「摂取カロリー」と「消費カロリー（基礎＋運動）」のバランスをリアルタイムで計算し、適正ペースかどうかを一目で判定します。

### 🤖 AIコンシェルジュ機能
* **データ分析アドバイス:** 今日の歩数やカロリー収支のデータをGemini AIが読み込み、あなた専用の的確なアドバイスを提供します。
* **性格変更機能:** 「優しいメイド」「ツンデレ幼馴染」「熱血コーチ」「クールな秘書」から、その日の気分に合わせてコンシェルジュの性格（口調）を変更できます。

### 📊 履歴・グラフ機能
* **ローカル自動保存:** 1日の記録をブラウザのローカルストレージに安全に保存します。
* **カラダの変化グラフ:** 過去の体重、BMI、基礎代謝の推移を美しい折れ線グラフ（Chart.js）で振り返ることができます。
* **履歴リスト:** 過去の歩数、消費カロリー、摂取カロリーなどの記録をリスト形式で確認可能です。

## 🛠️ 使用技術・システム連携
* **フロントエンド:** HTML5, CSS3, JavaScript (Vanilla)
* **地図描画:** Leaflet.js
* **ルート検索API:** OpenRouteService API
* **AI機能・画像解析API:** Google Gemini API (Gemini 2.5 Flash / Flash-Lite)
* **グラフ描画:** Chart.js
* **データ連携:** iOS「ショートカット」アプリによるヘルスケアデータ（歩数・階数）の自動取得
